## 中严重度问题详情

### C2: TBDMarkerChecker 中文词界问题

**问题**：正则 `(?i)\b(TBD|TODO|待定|待补充|未决定)\b` 中的 `\b` 是 ASCII 词界，不适用于中文。`"认证方式待定"` 中的"待定"前后是中文字符，`\b` 在中文字符边界的行为依赖于 Python re 引擎实现，可能匹配也可能不匹配。

**修复建议**：改用 lookaround 或直接去掉中文关键词的 `\b` 要求，改用 `(?:^|[^a-zA-Z])(TBD|TODO)(?:[^a-zA-Z]|$)|待定|待补充|未决定` 分别处理英文和中文标记。

**影响范围**：TBDMarkerChecker 对中文 SDD 的漏报风险。
