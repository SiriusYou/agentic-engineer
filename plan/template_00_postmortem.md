# Template 00 — 项目复盘（Phase 0 Postmortem）
# 用途：完成项目后回顾规划流程的有效性，收集改进数据
# 使用对话：任意
# 新建对话：不需要

> Phase 0 = 元工作流评估阶段。编号为 00 表示这是对整个 Step 1-5 流程本身的回顾。
> 推荐在每个项目完成后执行，非强制。

---

## 适用时机

- 项目完成后（推荐）
- 规划流程每执行一次后
- 建议在每步完成时记录耗时，复盘时补充其他字段

---

## 使用步骤

1. 复制本模板到 `spec/postmortem_v1.md`
2. 填写基本信息和各阶段耗时（壁钟时间为默认，活跃时间可选标注）
3. 填写收敛数据
4. 填写 Start/Stop/Continue 反馈
5. 为每个反馈点指定至少一个行动项
